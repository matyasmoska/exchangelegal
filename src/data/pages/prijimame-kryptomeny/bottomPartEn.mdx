# **We Accept Cryptocurrencies (Virtual Assets)**

For all the services we provide, you can pay not only in fiat currency but also with commonly used virtual assets – cryptocurrencies.

## **Which Virtual Assets Do We Accept?**

We accept **Bitcoin, Litecoin, Ether, Tether (USDT), Monero, and Dogecoin.**.

If you would like to pay using a different virtual asset, let us know — we’ll do our best to accommodate your request.
